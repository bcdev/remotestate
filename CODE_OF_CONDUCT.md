# Code of Conduct

This project is built for collaboration, learning, and practical work. The short version is simple: use common sense, 
be respectful, and help keep the space useful for everyone.

## Expected behavior

- Be considerate and constructive.
- Assume good intent when possible, and ask for clarification before escalating a misunderstanding.
- Give feedback about code and ideas, not about people.
- Keep communication clear, focused, and welcoming.
- Respect differing experience levels and backgrounds.

## Unacceptable behavior

- Harassment, threats, discrimination, or personal attacks.
- Doxxing, stalking, or sharing private information without permission.
- Sexualized language or imagery in project spaces.
- Spam, trolling, or repeated disruption.
- Deliberate attempts to intimidate, silence, or exclude others.

## Enforcement

Maintainers may remove or edit harmful content, close or lock threads, or refuse participation in project spaces when 
needed to keep the community safe and productive.

If you experience or witness behavior that violates this code, please report it privately through the issue tracker or 
by contacting [@forman](https://github.com/forman).
